build () line 95
Build the query with provided parameters. The function call can be put in a conditional statement to test whether your parameters can be used by the module to generate a meaningful query or not. After build call, you can call upon query to fetch the built query string.

Returns:

    boolean On successful build, returns true

Raises:

Invalid type of query parameters provided
